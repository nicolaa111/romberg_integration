Romberg Integration

Romberg Integration is a numerical method that improves trapezoidal rule accuracy using Richardson extrapolation.

Formula

R(i,j) = R(i,j-1) + (R(i,j-1) - R(i-1,j-1)) / (4^j - 1)

Example 1

Function: sin(x)

a = 0, b = 1

Result ≈ 0.45969769

Example 2

Function: sin(x)

a = 0, b = 3.1416

Result ≈ 2.0000
