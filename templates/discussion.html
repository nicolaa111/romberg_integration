Romberg Integration Method

Romberg Integration is a numerical method used to approximate definite integrals by improving the Trapezoidal Rule using Richardson extrapolation.

Formula

R(i,j) = R(i,j-1) + (R(i,j-1) - R(i-1,j-1)) / (4^j - 1)

Example 1: ∫ sin(x) dx from 0 to 1

Function: sin(x)

a: 0

b: 1

n: 4

Step 1: Initial Trapezoidal Rule

R(0,0) = (1/2)[sin(0) + sin(1)] = 0.5(0 + 0.84147) = 0.420735

Step 2: First Refinement

Midpoint: x = 0.5 sin(0.5) = 0.479425

Step 3: Improved Approximation

R(1,0) = 0.450079

Step 4: Romberg Extrapolation

R(1,1) = 0.45969769

Example 2: ∫ sin(x) dx from 0 to π

Function: sin(x)

a: 0

b: 3.1416

n: 4

Step 1: Initial Trapezoidal Rule

R(0,0) = (π/2)[sin(0) + sin(π)] = 0

Step 2: First Refinement

Midpoint: x = 1.5708 sin(1.5708) ≈ 1

Step 3: Improved Approximation

R(1,0) = 1.5708

Step 4: Romberg Result

R(1,1) = 2.0000
